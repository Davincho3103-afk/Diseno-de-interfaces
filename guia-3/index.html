📌 Problemática

Cuando un grupo de amigos intenta organizar una salida, suelen presentarse varios problemas:

  • Dificultad para coincidir fechas.
  • Desacuerdos sobre cuánto debe pagar cada persona.
  • Falta de claridad sobre quién ya pagó y quién no.
  • Desorden en los mensajes de WhatsApp.
  • No existe una herramienta sencilla enfocada exclusivamente en organizar salidas y dividir gastos automáticamente.

El proyecto busca resolver estos problemas mediante una aplicación que permita planificar eventos y calcular automáticamente la división de gastos según lo planeado.

🎯 Público Objetivo

  • Grupos de amigos.
  • Estudiantes universitarios.
  • Personas que organizan viajes o eventos pequeños.
  • Equipos de trabajo que realizan actividades sociales.

👤 Tipos de Usuario

  • Usuario creador: Puede crear grupos y eventos.
  • Usuario participante: Puede unirse a eventos y registrar pagos.
  • Administrador del grupo: Puede modificar gastos y confirmar pagos.

🔄 Entrada, Proceso y Salida del Sistema

Entrada

  • Datos de registro (nombre, correo, contraseña).
  • Creación de grupo (nombre, integrantes).
  • Creación de salida (fecha, descripción, presupuesto).
  • Registro de gastos (quién pagó, cuánto, concepto).
  • Confirmación de pagos.

Proceso

El sistema realiza:

  • Almacenamiento de datos en base de datos.
  • Cálculo automático del gasto total.
  • División equitativa del gasto entre participantes.
  • Cálculo de balances individuales.
  • Actualización del estado de pagos (pendiente / pagado).

Ejemplo:
Si se registra un gasto de $200.000 y hay 5 participantes, el sistema divide automáticamente $40.000 por persona. Si una persona pagó $120.000, el sistema calcula que le deben $80.000.

Salida

  • Calendario con salidas programadas.
  • Resumen total de gastos por evento.
  • Detalle de cuánto debe cada persona.
  • Estado de pagos.
  • Historial de eventos anteriores.

📌 Alcance del Sistema

  • Crear y administrar grupos.
  • Programar salidas con fecha.
  • Registrar y dividir gastos automáticamente.
  • Visualizar balances individuales.
  • Confirmar pagos realizados.
