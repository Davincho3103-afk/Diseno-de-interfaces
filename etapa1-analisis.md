SaliAmigos

David Alejandro Bonilla Mahecha
William David Vivas Maldonado

Cuando un grupo de amigos intenta organizar una salida, en algunas ocasiones suelen haber los siguientes problemas:
Dificultad para coincidir las fechas, 
desacuerdos sobre cuánto debe pagar cada persona, 
falta de claridad sobre quién ya pagó y quién no, 
mucho desorden a la hora de mandar mensajes en WhatsApp para organizar todo, 
no existe una herramienta sencilla y que este enfocada exclusivamente en organizar salidas entre amigos ademas de dividir gastos de forma clara y automática. 
El proyecto busca resolver estos problemas mediante una aplicación que permita planificar eventos y calcular automáticamente la división de gastos segun lo que se tiene planeado gastar.

El sistema está dirigido a:
Grupos de amigos.
Estudiantes universitarios.
Personas que organizan viajes o eventos pequeños.
Equipos de trabajo que realizan actividades sociales.
Tipos de usuario:
Usuario creador: Puede crear grupos y eventos.
Usuario participante: Puede unirse a eventos y registrar pagos.
Administrador del grupo: Puede modificar gastos y confirmar pagos.

Entrada, Proceso y Salida del Sistema

Entrada
-Datos de registro del usuario (nombre, correo, contraseña).
-Creación de grupo (nombre del grupo, integrantes).
-Creación de salida (fecha, descripción, presupuesto opcional).
-Registro de gastos (quién pagó, cuánto pagó, concepto).
-Confirmación de pagos entre usuarios.

Proceso
El sistema realiza:
-Almacenamiento de datos en base de datos.
-Cálculo automático del gasto total.
-División equitativa del gasto entre los participantes.
-Cálculo de balances individuales (cuánto debe o cuánto le deben).
-Actualización del estado de pagos (pendiente / pagado).

Ejemplo de proceso:
Se registra un gasto de $200.000 y hay 5 participantes. El sistema divide automáticamente: $40.000 por persona. Si una persona pagó $120.000 el sistema debe calcular y decir que le deben $80.000.

Salida
Calendario con salidas programadas, 
Resumen total de gastos por evento, 
Detalle de cuánto debe cada persona, 
Estado de pagos, 
Historial de eventos anteriores.

Alcance del Sistema
El sistema permitirá: crear y administrar grupos, Programar salidas con fecha, registrar y dividir gastos automáticamente, visualizar balances individuales, confirmar pagos realizados.

El sistema NO incluirá en esta primera versión:

Integración con bancos o pagos automáticos.

Geolocalización en tiempo real.

Red social pública para conocer desconocidos.

Sistema complejo de reputación o verificación avanzada.
